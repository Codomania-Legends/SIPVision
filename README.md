<h1>📈 SIPVision:</h1> 
Smart Investment Goal Planner</br>
SIPVision is an intuitive tool designed to help investors visualize their wealth creation journey. It provides deep insights into Systematic Investment Plans (SIPs) by factoring in inflation, real returns, and dynamic "what-if" scenarios. 💸✨</br>

## ✨ Key Features

| Feature | Description |
| ------- | ----------- |
| 📊 **Interactive Charts** | Dynamic line and doughnut charts powered by **Chart.js** to visualize wealth gain vs. principal. |
| 🧮 **Precision Math** | Toggle between **SIP (Monthly)** and **Lumpsum** modes with real-time compounding logic. |
| 🎈 **Inflation Adjuster** | See your "Future Value" in "Today's Purchasing Power" to keep your goals realistic. |
| 🎯 **Goal-Based Tracking** | Input a target amount, and let SIPVision calculate the required monthly investment. |
| 🌓 **Glassmorphism UI** | A sleek, modern dashboard designed with **Tailwind CSS** for a premium financial feel. |
| 📱 **Mobile Optimized** | Check your projections on the go with a fully responsive layout. |

## 🛠️ Tech Stack

  <img src="https://skillicons.dev/icons?i=react,vite,tailwind,js,figma,netlify&perline=6" alt="Tech Stack Icons" />

<br/>

- **Frontend:** React.js (Vite)
- **Data Visualization:** Chart.js / React-Chartjs-2
- **Styling:** Tailwind CSS (Modern FinTech Design)
- **State Management:** React Hooks (useState, useEffect)
- **Deployment:** Vercel / Netlify

🛠️ Setup Instructions</br>
Follow these steps to run SIPVision locally:</br>

1. Clone the Repository:</br>
  git clone https://github.com/Codomania-Legends/SIPVision.git</br>
  cd SIPVision</br>
2. Install Dependencies:</br>
    npm install</br>
4. Run Development Server:</br>
   npm run dev</br>
6. Build for Production:</br>
   npm run build</br>
